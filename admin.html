⚙️ Setup Guide

Step 1 — Create Google Sheet

1. Go to sheets.google.com and create a new spreadsheet
2. Name it: ExamCracker Database
3. Create 3 sheets (tabs at bottom):
  → Questions — stores all question data
  → Keys — stores access keys
  → Results — stores student results
4. Copy the Spreadsheet ID from the URL (the long string between /d/ and /edit)

Step 2 — Sheet Column Headers

Add these exact headers in Row 1 of each sheet:

Questions Sheet — Row 1:

Question_ID | Subject | Year | Question | Option_A | Option_B | Option_C | Option_D | Correct_Answer | Explanation | Difficulty | Paper

Note: Correct_Answer = 0 for A, 1 for B, 2 for C, 3 for D

Keys Sheet — Row 1:

Key | Status | Purchase_Date | Student_Name | Attempts | Expiry_Date

Results Sheet — Row 1:

Key | Test_Name | Date | Total_Questions | Correct | Wrong | Unattempted | Score | Percentage | Time_Taken

Step 3 — Deploy Google Apps Script

1. Go to script.google.com → New Project
2. Paste the entire Code.gs file into the editor
3. Replace YOUR_SHEET_ID_HERE with your actual Sheet ID
4. Click Deploy → New Deployment → Web App
5. Set: Execute as Me, Who has access: Anyone
6. Click Deploy, copy the Web App URL
7. Paste that URL in index.html as the value of APPS_SCRIPT_URL

Step 4 — Connect Website to Apps Script

In index.html, find this line near the top of the <script> section:
const APPS_SCRIPT_URL = ''; Replace the empty string with your deployed Web App URL:
const APPS_SCRIPT_URL = 'https://script.google.com/macros/s/YOUR.../exec'; Once set, key validation and result saving will use Google Sheets live.

Step 5 — Deploy on GitHub Pages

1. Create a free GitHub account at github.com
2. Create a new repository named: examcracker
3. Upload your files: index.html and admin.html + your Image1.png, Image2.png, Image3.png
4. Go to repository Settings → Pages → Branch: main → Save
5. Your site will be live at: https://yourusername.github.io/examcracker

Step 6 — Your WhatsApp Payment Flow

1. Student clicks Subscribe ₹49 on the website
2. Modal shows your UPI ID and WhatsApp number
3. Student pays ₹49 → sends screenshot on WhatsApp
4. You confirm payment → open this admin.html
5. Go to Generate Key → enter student name → click Generate
6. Copy the key → send on WhatsApp
7. Student enters key on website → all papers unlock ✅

⚡ Update these in index.html before going live:
  • UPI ID in the subscribe modal
  • WhatsApp phone number in the subscribe modal
  • APPS_SCRIPT_URL variable
  • Replace Image1.png, Image2.png, Image3.png with real images
